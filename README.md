# start

## 用到的命令

- 切换进程
  - fg %pid

- 远程同步本地
  - git fetch / git pull

- 本地同步给远程
  - git push

- 创建标签
  - git tag -a

- 查看标签
  - git tag -l

- 同步到远程tag
  - git push origin --tag

- 提交代码

```test
   rust-test1 on  main [!] is 📦 v0.1.0 via 🦀 v1.77.2
    ❯ git checkout -b my_test/fix-r1
    Switched to a new branch 'my_test/fix-r1'

    rust-test1 on  my_test/fix-r1 [!] is 📦 v0.1.0 via 🦀 v1.77.2
```
